# EuroClub App

## Запуск
```
npm install
npm run dev
```
Відкрийте http://localhost:3000

## Деплой на Vercel
```
npx vercel
```

## API
Тестовий ключ: apitest_joieerv65v6j
URL: https://eclub.com.ua/api/v1/json/[метод]/[ключ]/

## Екрани
1. Головна — пошук рейсів
2. Результати пошуку — реальний API
3. Бронювання — форма пасажирів + місця + послуги
4. Вибір місця — інтерактивна схема автобуса
5. Підтвердження замовлення
6. Оплата — вибір методу
7. Мої квитки — збережені локально
8. Профіль
